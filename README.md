# VoiceGift — лендинг «Закажите песню или медитацию для ребёнка»

Простой статический сайт, который развёрнут на GitHub Pages.

## Структура

- `index.html` — разметка лендинга.
- `styles.css` — стили (адаптив, анимации, визуальный стиль).
- `script.js` — логика динамической цены, скролла и заглушки отправки формы.

## Локальный запуск

Откройте `index.html` в браузере (достаточно двойного клика или `Open with Live Server` в редакторе).

## GitHub Pages

Сайт публикуется из ветки `main` из корня репозитория.
